# Backend-ScryX

## Crie uma Virtual Enviroment - virtualenv
```
python3 -m venv venv
```

## Importe os requisitos
```
pip install -r requeriments.txt
```

## Inicie o Banco de Dados MongoDB - Porta 27017

## Inicie o servidor
```
python3 backend.py
``` 

## Inicio o servidor de cotações
```
python3 research_assistant.py
```
